# SRA read

A library that downloads NCBI SRA runs in FASTA format, using [NCBI NGS-SDK](https://github.com/ncbi/sra-tools/wiki/09.-Downloading-NGS-SDK).

## Build

To build the library run `make`.

## Demo

A small demo program that demonstates how to use the library is in `demo.c` file. To build it run `make demo`.
